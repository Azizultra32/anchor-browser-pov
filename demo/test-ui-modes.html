UI Modes Test - Phase 3 Checkpoint

Testing Clinician vs Debug UI modes in Ghost Overlay

Test Instructions:

  1. Click the "Ghost" button to open the overlay
  2. Click "Map" to map the form fields
  3. Toggle between "Clinician" and "Debug" modes using the Mode button
  4. Observe the differences in field display

Clinician Mode:

  • Shows only field labels
  • Hides technical details (selectors, roles)
  • Simplified controls (Map only)
  • Clean, minimal interface

Debug Mode:

  • Shows all field metadata
  • Displays CSS selectors
  • Shows field roles
  • All controls visible (Map, Send, Fill)
